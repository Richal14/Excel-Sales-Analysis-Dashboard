# Excel Sales Analysis Dashboard

## Project Overview
This project analyzes sales data using Microsoft Excel to identify trends, regional performance, and category-wise contribution.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- KPIs
- Dashboard Design

## KPIs Created
- Total Sales
- Total Profit
- Total Quantity
- Average Order Value

## Analysis Performed
- Sales by Region
- Sales by Category
- Sales Trend Over Time
- Top Products by Profit
- Sales by City

## Key Insights
- Identified top-performing regions and categories
- Analyzed monthly sales trends
- Evaluated profitability across products

- Sales_Analysis_Dashboard.xlsx
